# Godot Game Tools #
This is a blender add-on that contains tools to ease the creation/import process of assets into Godot Game Engine
https://github.com/godotengine/godot

### How To Use ###
https://www.youtube.com/watch?v=fzKitRyqAK0&t=79s

### Root Motion Sample Project ###
GGT Github Repository comes packed with a sample root motion project contributed by @chrisphyffer
You can use this demo project to set up your exported rootmotion animations with Godot's AnimationTree and BlendSpaces
This project even has treatments for console joypad support, such as Playstation or Xbox Controllers.

### Features ###
- Mixamo Utilities (Rebuilt With Easier and Simpler Workflow)
- Texture Controls (Still Work-In-Progress, right now compatible with Mixamo textures to export)
- BVH Manager (Missing Export/Import features - Work-In-Progress)
- Tileset Generator (Fully Compatible with Godot Export)
